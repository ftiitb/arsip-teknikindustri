# Arsip Teknik Industri

Aplikasi Sistem Informasi Arsip untuk pengelolaan surat masuk, surat keluar, dan disposisi.

## Teknologi

Aplikasi ini dibangun menggunakan:
- **Backend**: PHP Native
- **Frontend**: ExtJS Framework
- **Database**: MySQL
- **Libraries**:
  - ADODB
  - FPDF
  - PHPExcel
